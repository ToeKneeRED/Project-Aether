# Project Aether

This repository contains the **source files** for Project Aether.

The project is shared so others can:
- download the project
- explore or mod the content
- suggest changes or improvements

This is **not a packaged build,** it is the full project.

## Requirements
- **Unreal Engine 5.5.4**
- **A functioning frontal lobe**

Make sure you open the project using **UE 5.5.4**, as other versions may cause compatibility issues.
